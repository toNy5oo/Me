To be or not to be

Being a web developer nowadays is a challenge more than ever. With technology improving daily, offering many different languages, frameworks, libraries and various tools of development, navigating this ocean of knowledge is not easy task.

What makes then a developer, a good one?

Problem solving forma mentis, good social and human skills, ability to acknowledge criticism in a constructive way, are some of the keys to be a great developer. Writing code is essential, but that can be done by many. Refining not only the way you code but as well the attitude towards your colleagues and the mindset towards learning new skills, would make a great asset for any company.
